===================
Watchface Gradient
===================

The watchface gradient widget allows you to create a pointer watchface with a scanning effect for each pointer.

Usage
-----

Create Widget
~~~~~~~~~~~~~~
To create a new watchface gradient widget, you can call the function :cpp:any:`gui_watchface_gradient_create` to achieve it.

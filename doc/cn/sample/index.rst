=======
示例
=======

为了帮助用户熟悉使用该环境，我们提供了一些示例应用程序。这些示例程序会不断增加，您可以从以下配置中选择适合您的配置，配置文件为``menu_config.h``。


.. figure:: https://foruda.gitee.com/images/1718765392647198639/ac4aa231_13408154.png
   :align: center
   :width: 800px

   配置选择


每个应用程序的入口点如下：

.. code-block:: c

    GUI_INIT_APP_EXPORT(app_init);
